# GameNative Source Code Analysis: Config & API Usage Findings

This report details the various online APIs and GitHub repositories used by the GameNative application to download game configurations, chip-specific settings, drivers, and other components.

---

## 1. Game & Chip Specific Configurations (Best Config Service)
This is the primary mechanism for optimizing games for specific GPUs (chips).

*   **API URL:** `https://gamenative-best-config-worker.gamenative.workers.dev/api/best-config`
*   **Source File:** `app/src/main/java/app/gamenative/utils/BestConfigService.kt`
*   **Description:** Sends the `gameName` and `gpuName` (GPU Renderer string) to a Cloudflare Worker. It returns a JSON configuration containing optimal settings such as:
    *   Container variant (Bionic/Glibc)
    *   Graphics driver and version
    *   DXVK/VKD3D versions and configurations
    *   Wincomponents and Environment variables
    *   Box64/FEXCore presets
*   **Usage:** Automatically applied during container setup or manually via "Use Known Config" in the game menu.

## 2. Component Manifests & Repositories
The app uses several manifests to track and download external components like drivers and Wine versions.

*   **Main Component Manifest:**
    *   **URL:** `https://raw.githubusercontent.com/maxjivi05/GameNative-Performance/refs/heads/main/manifest.json`
    *   **Source File:** `app/src/main/java/app/gamenative/utils/ManifestRepository.kt`
    *   **Description:** Central manifest for all downloadable components.

*   **MTR (Mesa Turnip) Components:**
    *   **Manifest URL:** `https://raw.githubusercontent.com/maxjivi05/Components/main/contents.json`
    *   **Drivers API:** `https://api.github.com/repos/maxjivi05/Components/contents/Drivers`
    *   **Source Files:** `ContentsManagerDialog.kt`, `ComponentsManagerDialog.kt`
    *   **Description:** Lists available content profiles and drivers for Adreno GPUs.

*   **GameNative Landing Page (Driver Manifest):**
    *   **URL:** `https://raw.githubusercontent.com/utkarshdalal/gamenative-landing-page/refs/heads/main/data/manifest.json`
    *   **Source File:** `ComponentsManagerDialog.kt`
    *   **Description:** Used to discover and download drivers from the official landing page repository.
